CALCULADORA CONVERSORA DE TEMPERATURAS


    Esta Api permite al usuario realizar conversiones entre temperaturas en grados Celsius y Fahrenheit. Esta puede ser usada desde JavaScript, el navegador, de forma local y desde Java
  • Empecemos explicando cómo podriamos conectarnos desde JavaScript: Esta API se encuentra desplegada en la siguiente dirección: https://calm-earth-36734.herokuapp.com
    • Para conectar desde un archivo .JS basta con utilizar la ruta correspondiente según la conversión que se desee hacer y realizar la petición POST. En esta caso se utilizara la libreria axios para poder realizar la petición http (de igual forma se puede usar XMLHttpRequest(), fetch, etc).

  • Celsius a Fahrenheit :

    'https://calm-earth-36734.herokuapp.com/convert/toFahrenheit/'

  • Fahrenheit a Celsius :

    'https://calm-earth-36734.herokuapp.com/convert/toCelsius/'

....De esta forma la conversión que se necesitaba quedará guardada en la variable que recibe la respuesta para ser usada como se desee.

  • Para usar esta API desde el navegador de manera local, basta con descargar el repositorio donde se encuentran los archivos fuente de esta API. Luego de obtener el repositorio se debe acceder a la dirección desde su navegador de preferencia: http://localhost:4567.

    Repositorio:     Ir a github
    • En la parte posterior de la página mostrada encontrada los espacios para realizar las conversiones. Ingrese el valor o valor deseado en los espacios indicados y oprima el botón para obtener los resultados.
    De esta manera se pueden realizar las conversiones empleando accediendo a la API de manera local.

    Para emplear la API sin interfaz gráfica podemos simplemente agregar los valores a convertir según la ruta de la siguiente manera:

    http://localhost:4567/convert/toCelsius/"valor"

    http://localhost:4567/convert/toFahrenheit/"valor"
